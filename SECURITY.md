# Security Policy

## Supported Versions

House Hunt Kisii is under active development. Security fixes are applied to the `main` branch only.

| Version | Supported |
| ------- | --------- |
| main    | ✅        |

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not** open a public issue.

Instead, report it privately by opening a [GitHub Security Advisory](https://github.com/learninghub44/HOUSE-HANTING/security/advisories/new) on this repository, or by contacting the maintainer directly via [learninghub44](https://github.com/learninghub44).

Please include:

- A description of the vulnerability and its potential impact
- Steps to reproduce it
- Any suggested remediation, if you have one

We aim to acknowledge reports within a few days and will keep you updated as the issue is triaged and resolved. Please allow a reasonable time for a fix to be released before any public disclosure.

## Scope

This policy covers the House Hunt Kisii application code in this repository. It does not cover third-party services it integrates with (Groq, Cloudflare, Supabase, Didit, PayHero) — please report issues in those services to their respective owners.
